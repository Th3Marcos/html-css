Representação das Cores

As cores podem ser representadas de várias formas, incluindo:

  • Nome da Cor: Exemplo: "vermelho", "azul", "verde".
  • Hexadecimal: Exemplo: "#FF0000" para vermelho, "#0000FF" para azul.
  • RGB: Exemplo: "rgb(255, 0, 0)" para vermelho, "rgb(0, 0, 255)" para azul.
  • HSL: Exemplo: "hsl(0, 100%, 50%)" para vermelho, "hsl(240, 100%, 50%)" para azul.

Essas representações são amplamente utilizadas em design gráfico, desenvolvimento web e outras áreas relacionadas às artes visuais.
