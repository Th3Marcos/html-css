Listas

Listas Ordenadas

Listas Ordenadas, são as listas cuja ordem de seus itens é muito inportante. Elas usam as tags <ol> com fechamento </ol> para delimitar a lista e <li> para cada item da lista. No inglês Ordered List e List Item.

Ex.:

  1. Item 1
  2. Item 2
  3. Item 3

Listas não Ordenadas

As Unordered Lists são as Listas não Ordenadas, são como as anteriores, porém sua ordem não influencia no seu significado.

Elas usam as tags <ul> para delimitar a lista e a tag <li> para seus itens.

Ex.:

  • Item 1
  • Item 2
  • Item 3

Misturando Tudo

Você pode misturar as duas para criar listas mistas.

Ex.:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
    • Item 3.1
    • Item 3.2
    • Item 3.3

Listas de Definição

São como um dicionário, onde cada termo é definido.

Usam as tags <dl> (definition list) para demarcar a lista, seu termo fica na tag <dt> (definition term) e sua descrição na tag <dd> (definition description).

Ex.:

HTML
Linguagem de marcação para criar páginas web.
CSS
Linguagem de estilo para definir a aparência das páginas web.
JavaScript
Linguagem de programação para adicionar interatividade às páginas web.
